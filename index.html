Visualizations of Gender Wage Gap in the US¶

Xuan Bu

The wage gap between genders is a public issue that attracts widely concerns and discussions. Over the decades, policies aimed at narrowing the gender wage gap are numerous, which brings about the question: have the wage gaps between genders narrowed after these efforts?

The project will explore the gender wage gap and its trends from the perspectives of race, occupation, age, and geography, education etc. Although the project focuses on the gender wage gap in the US, meanwhile, it compares the gender wage gap of the US with the gender wage gaps of other countries over the years.

#@title SHOW CODE
# create dataset
wage_gap_gender = gap_age_gender_race[gap_age_gender_race["Characteristic"]=="Median weekly earnings"]

# using length to visualize the wage gap between genders
# visual encoding used: length; position
line_gap_gender = alt.Chart(wage_gap_gender).mark_line().encode(
    x = alt.X("year:O", title = "year"),
    y = alt.Y("Total:Q", title = "wage gap"),
    detail = "year:O")

# using bar to represent the wage gap between genders by year
# visual encoding used: color saturation; position
gap_diff = alt.Chart(wage_gap_gender).mark_area(color='grey', opacity=0.7).encode(
    x = "year:O",
    y = "Gender Diff:Q"
)

# Add points for wage gap between genders
# visual encoding used: color hue; position; shape
point_gap_gender = alt.Chart(wage_gap_gender).mark_point(
    size = 50,
    opacity = 1,
    filled=True).encode(
        x = "year:O",
        y = "Total:Q",
        color = alt.Color("Gender:N", scale=alt.Scale(range=["#60acfc", "#96dc88"])),
        shape = "Gender:N").properties(
    title={
      "text": ["Gender Wage Gap in the US from 1994 to 2018"], 
      "subtitle": ["Comparing the median weekly earnings between genders over the past 20 years"],
      "color": "Black",
      "subtitleColor": "Black"
    })


# combine all charts
alt.layer(
    line_gap_gender,
    point_gap_gender,
    gap_diff
).properties(height=300, width=600
).configure_title(fontSize=16, font='Helvetica Neue', orient="top", anchor="middle").configure_legend(titleColor='black', titleFontSize=12)

# create dataset
url_gap_gender_geo = "https://raw.githubusercontent.com/XuanBu21/capp-30239-w20-A01/master/data/2017-sex-state.csv"
gap_gender_geo = pd.read_csv(url_gap_gender_geo)

# load the geojson file and connect it with the dataframe of gender wage gap by states.
states = alt.topo_feature(data.us_10m.url, feature='states')

# using the choropleth map to illustrate the distribution of gender wage gap among different states
# visual encoding used: position; density; color hue; connection
geo = alt.Chart(states).mark_geoshape().encode(
    color=alt.Color('Earnings Ratio:Q', legend=alt.Legend(format=".0%"))
).transform_lookup(
    lookup='id',
    from_=alt.LookupData(gap_gender_geo, 'id', ['Earnings Ratio', 'Male', 'Female', 'State'])
).project(
    type='albersUsa'
).properties(
    width=580,
    height=350
)

label = alt.Chart(gap_gender_geo).mark_text(color="#33334d", font='Helvetica Neue', opacity=0.6).encode(
    longitude='longitude:Q',
    latitude='latitude:Q',
    text='state:N',
    tooltip=['Male', 'Female', 'Earnings Ratio']).properties(
    title={
      "text": ["Gender Wage Gap by States in the US in 2017"], 
      "subtitle": ["Comparing the median yearly earnings by gender across the states of the US",
                   "*Earnings Ratio = female-to-male median yearly earnings"],
      "color": "Black",
      "subtitleColor": "Black"
    }
)

alt.layer(
    geo,
    label
).configure_title(fontSize=16, font='Helvetica Neue', orient="top", anchor="middle").configure_legend(titleColor='black', titleFontSize=12)

# using area map to illustrate the gender wage gap among different education level group

    x=alt.X("education:N", sort=['Less than high school', 'High school', "Associate's degree", "Bachelor's degree", "Graduate"], axis=alt.Axis(labelAngle=0)),

    color = alt.Color("gender:N", title="Gender", scale=alt.Scale(range=["#60acfc", "#96dc88"]))
).properties(
    width=600,
    height=350
)


text = alt.Chart(gap_gender_edu).mark_text(
    color='#302d2d',
    opacity=0.9,
    baseline='middle',
    dx=5,
    dy=-35
).encode(
    x=alt.X("education:N", sort=['Less than high school', 'High school', "Associate's degree", "Bachelor's degree", "Graduate"]),
    y=alt.Y('gap:Q', title='income'),
    detail='earning ratio',
    text=alt.Text('earning ratio', format=".0%")
)


rule = alt.Chart(gap_gender_edu).mark_rule(color='red').encode(
    y='mean(income):Q'
)

    area,
    text,
    rule
).properties(
    title={
      "text": ["Gender Wage Gap by Education Level in the US in 2017"],

The graphics illustrates how many times an occupation has been ranked the largest/smallest ER (Earning Ratio) occupation among states.

As shown in the figure, computer, engineering, and science" occupations have the highest ER in 26 states, that is, computers-related occupations have the lowest gender wage gap in more than half of the states. "Production, transportation, and material moving" occupations, on the other hand, have the largest gender wage gap in more than a third of states. "Sales" and "service" occupations are also the industries with comparable big gender wage gap. It is worth noting that the "management, business, and financial" occupations are the ones with stable gender wage gap in the United States-being ranked the occupations with the biggest ER only in one state.
